# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for Dagpedia,
following the [MADR](https://adr.github.io/madr/) format.

An ADR documents a significant decision made during the development of Dagpedia,
including the context, the options considered, and the rationale for the choice.
